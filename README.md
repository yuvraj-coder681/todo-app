# ToDo App

A simple web-based Todo App where users can:

- Add tasks
- View the task list
- Track their tasks

## Tech Stack

- HTML
- CSS
- JavaScript

## How to Use

1. Type a task in the input field
2. Click on **"Add Task"** button
3. Your task will appear in the list below
